# Vallecito

Trekking de ~40min pela Cordilheira de Sal + visita ao Magic Bus (ônibus
abandonado icônico).

- Saída: 15h30–16h | Retorno: 19h30
- Altitude: 2.250m (baixa) | Distância: 50km
- Inclui: transporte, guia
- Preço: ver `04_precos/precos_2026_atacama.md`

## Para quem é

Quem gosta de caminhada leve e fotos icônicas; alternativa de baixa altitude
boa para os primeiros dias.

## Quando escalar

Mobilidade reduzida (trekking) → confirmar com humano se adaptável.
