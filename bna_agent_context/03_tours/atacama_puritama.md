# Termas de Puritama

Piscinas naturais termais de 25°C a 30°C no meio do deserto.

- Tour manhã: saída 8h / retorno 13h | Tour tarde: saída 13h / retorno 18h
- Altitude: 3.500m | Distância: 60km
- Inclui: transporte, entrada do parque
- Preço: ver `04_precos/precos_2026_atacama.md`

## Para quem é

Relaxamento + "passeio com banho" (junto com Cejar e Baltinache é a resposta
para "tem passeio com água/banho?"). Bom para casais e famílias (atenção à
altitude 3.500m para crianças pequenas — ver saúde).

## Quando escalar

Restrições de saúde relevantes em altitude 3.500m → handoff.
