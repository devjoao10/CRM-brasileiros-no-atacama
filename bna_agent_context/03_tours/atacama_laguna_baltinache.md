# Lagunas Escondidas de Baltinache

7 lagoas azul-turquesa, caminhada pelo salar, flutuação. Tour aventura
(acesso mais difícil).

- Tour manhã: saída 7h30 / retorno 13h | Tour tarde: saída 13h30 / retorno 19h
- Altitude: 2.500m | Distância: 120km
- Inclui: transporte, guia, café (manhã) ou coquetel (tarde), entrada
- Preço: ver `04_precos/precos_2026_atacama.md`

## Para quem é

Quem quer flutuação com clima mais aventureiro e menos gente que a Cejar.

## Quando escalar

Mobilidade reduzida (caminhada no salar) → confirmar com humano.
