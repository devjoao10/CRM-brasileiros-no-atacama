# Restrições e cuidados de saúde — tabela de referência

> Consolidação das duas fontes. Regras de idade = LIVE (vigentes); demais =
> export de maio `[PENDENTE_VALIDACAO]`.

| Perfil | Restrição | Ação da BIA |
|---|---|---|
| Criança < 7 anos | Proibido acima de 4.000m | Avisar, sugerir alternativas baixas/privado |
| Criança < 12 anos | Não recomendado Uyuni; evitar alta altitude | Avisar, sugerir alternativas |
| Idoso 65+ | Autorização médica + termo | Sugerir privado; handoff p/ avaliação |
| Gestante | Não recomendado acima de 3.500m; autorização médica | Handoff |
| Cardíaco / hipertenso / diabético / epilético / asmático | Não recomendado acima de 3.500m; liberação médica | Handoff |
| Mobilidade reduzida | Trekkings/caminhadas podem não ser adaptáveis | Handoff p/ confirmar viabilidade |
| Trekking montanha (Lascar/Toco) | 16–65 anos; guia pode vetar; resgate por conta do passageiro | Informar limites; handoff se dúvida |

## Outros cuidados

- Seguro viagem internacional: fortemente recomendado (LIVE) — a BIA reforça
  mas não vende seguro.
- Casos excepcionais: avaliados pela consultora humana — nunca prometidos
  pela BIA.
