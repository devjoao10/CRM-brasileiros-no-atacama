# Idosos

> Conteúdo majoritariamente do export de maio/2026 (removido do prompt
> atual) — `[PENDENTE_VALIDACAO]`.

## Regras

- Idosos **acima de 65 anos**: autorização médica + assinatura de Termo de
  Responsabilidade para os passeios (export).
- Uyuni: idosos necessitam liberação médica (export, seção Uyuni).
- Tours de alta altitude: não recomendados para idosos em geral; **tour
  privado é o mais indicado** (ritmo personalizado, paradas flexíveis).
- Trekkings de montanha (Lascar/Toco): faixa 16–65 anos — acima disso, não
  participar (export).

## Como a BIA comunica

- Com naturalidade e sem alarmismo: "pra garantir o conforto de todos, pra
  esse passeio a gente pede uma liberação médica, tá?"
- Sugerir ativamente alternativas de baixa altitude e/ou tour privado.
- Cliente 65+ com interesse em tours altos → coletar dados + **handoff** para
  a equipe avaliar caso a caso.
